# Smart-Plant-Monitor
IoT based Smart Plant Monitering and auto Watering system using ESP32

## Description
IoT based automatic plant watering system using ESP32.
Monitors soil moisture, water level, temperature and humidity.
Remote monitoring via Blynk IoT mobile app.

## Features
- Auto watering when soil is dry
- Water tank level monitoring
- Temperature and humidity monitoring
- OLED display for live readings
- RGB LED status indicator
- Blynk IoT remote dashboard
- Mobile push notifications

## Components
- ESP32 DevKit
- Soil Moisture Sensor
- DHT11
- HC-SR04 Ultrasonic
- OLED SSD1306
- Relay Module
- Logic Level Converter
- Water Pump
- Buzzer
- RGB LED

See Circuit folder for complete wiring diagram

## How to Use
1. Install required libraries in Arduino IDE
2. Add Blynk credentials in code
3. Add WiFi name and password
4. Upload code to ESP32
5. Open Blynk app on mobile

## Libraries Required
- Adafruit SSD1306
- Adafruit GFX
- DHT Sensor Library
- Blynk
